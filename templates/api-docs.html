SCABLE API Documentation

1. Generate SBOM and Trace Reachable Components (`/sbom`)

This API provides SBOM in three standard formats: CycloneDX, SWID, and SPDX. Additionally, it traces "Reachable Components" affected by the user input and visualizes the analysis results on a web page.

HTTP Request

curl "http://scable.kr:8282/sbom?repo_url={repo-url}&lan=python"

Request Parameters

Parameter Type Required Description
repo_url string Yes The URL of the repository to analyze
lan string Yes The language of the repository (fixed value: python)

2. Check Package Reputation (`/package-check`)

This API checks the reputation of a PyPI package. It evaluates factors such as the number of downloads, GitHub stars, and the potential for typosquatting.

HTTP Request

curl "http://scable.kr:8282/package-check?package_name={package-name}"

Request Parameters

Parameter Type Required Description
package_name string Yes The name of the PyPI package to check

3. Settings (`/settings`)

This API allows you to configure settings such as GitHub integration, reputation thresholds, and packages to skip for reputation checks.
